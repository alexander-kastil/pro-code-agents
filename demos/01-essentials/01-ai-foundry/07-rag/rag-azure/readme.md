# Build a custom chat app with the Azure Al Foundry SDK

[Build a custom chat app with the Azure Al Foundry SDK](https://learn.microsoft.com/en-us/azure/ai-studio/tutorials/copilot-sdk-create-resources?tabs=windows)

Test with prompt:

```bash
python chat_with_products.py --query "I need a new tent for 4 people, what would you recommend?"
```

Test with logging:

```bash
python chat_with_products.py --query "I need a new tent for 4 people, what would you recommend?" --enable-telemetry
```
