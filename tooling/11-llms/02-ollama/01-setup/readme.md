# Ollama

- Setup Ollama in WSL
- Setup Ollama UI

## Setup Ollama in WSL

- Install in WSL:
    
    ```bash
    curl https://ollama.ai/install.sh | sh
    ```

- Run Ollama in WSL:

    ```bash
    ollama run <model>
    ollama run mistral
    ```    

- Get help on Ollama in WSL:

    ```bas
    ollama --help
    ```
